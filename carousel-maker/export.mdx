When the deck is ready, the top-right of the editor has everything you need to ship it. There are four paths out of the editor — pick the one that matches where the carousel is going.

/>

## Which option to use

| You want to… | Use |
| --- | --- |
| Post the carousel on LinkedIn now or schedule it | **Publish** |
| Upload to LinkedIn yourself, or save the file for later | **Download › Download PDF** |
| Post on Instagram, X, a newsletter, or a blog | **Download › Download Images** |
| Re-use this deck as a starting point next time | **⋯ menu › Save as Template** |

## Download PDF

A single PDF with every slide in order — this is the format LinkedIn uses for native carousel posts. Click **Download › Download PDF** and the file drops into your browser's downloads folder.

<Tip>
  Posting on LinkedIn yourself? Start a new post, click the **Document** icon, and pick the PDF. LinkedIn renders it as a swipeable carousel automatically.
</Tip>

## Download Images

A zip with one image per slide. Use this for any platform that takes individual images instead of a PDF — Instagram carousels, X threads, newsletters, blog embeds. Slide order is preserved in the filenames.

## Publish to LinkedIn

Click **Publish** and Carousify hands the deck to the **Post Generator** — the same composer you'd use for any LinkedIn post. From there you write the caption, pick the LinkedIn account, and either post now or schedule for later.

<Card title="Schedule a post" icon="calendar" href="/scheduling/schedule-post">
  Walk through the post composer — caption, account, and scheduling.
</Card>

## Save as Template

If this deck is one you'll want to start from again, open the **⋯** menu next to **Publish** and pick **Save as Template**.

<img
  src="/images/carousel-maker/publish-menu.png"
  alt="More menu next to Publish showing Delete Carousel and Save as Template"
  className="rounded-xl border"
/>

It lands at the top of **Carousels › Templates**, above the Carousify-designed library. Open it like any other template and you're back in the editor.

<Note>
  Templates and drafts are workspace-scoped. Saving a template in one workspace doesn't make it available in another — save it again in each workspace where you'd use it.
</Note>
