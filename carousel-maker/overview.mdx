The Carousel Maker is Carousify's visual editor for LinkedIn carousel posts. You can start from a blank canvas, drop in one of the pre-built templates, or let **Smart Carousel** generate a full deck from a topic, link, or YouTube video. Whatever path you start on, you end up in the same editor — slides on the left, live preview on the right, brand kit one click away.

## Where to find it

In the left sidebar, open **Carousels › Create New** to start fresh, or **Carousels › Templates** to browse the gallery first. Existing work-in-progress lives under **Carousels › Drafts**.

## What the editor gives you

- **Sizes** — switch between **Portrait** (4:5) and **Square** (1:1) at any time. The whole deck reflows to the new ratio.
- **Slide tools** — drag-and-drop to reorder, **Reset Slide** to revert one back to its starting state, and slide-by-slide navigation at the bottom.
- **Elements** — headings, subheadings, body text, images (PNG/JPG/GIF up to 10MB), plus shapes, icons, and decorative elements via **Add Elements**.
- **Tabs** — the editor groups its panels into:
  - **Content** — type and design the current slide.
  - **Global** — settings that apply across every slide (background, default fonts, footer).
  - **Brand Kit** — colors, fonts, and logos saved at the workspace level.
  - **Preview** — see how the deck will look on LinkedIn before exporting.
- **Focus Mode** — strips the chrome away when you want to design without distractions.
- **Save Draft / Download / Publish** — top-right at all times. Drafts auto-save as you work, **Download** exports as PDF or PNG set, and **Publish** hands the deck off to the post composer.

## Brand Kit

Every workspace has a Brand Kit — saved colors, fonts, and logos that apply across every carousel you build. Pick the **Brand Kit** tab in the editor to switch between kits or pull the current one's palette into the slide you're working on.

## Keyboard shortcuts

The editor is built around shortcuts — press **?** anywhere in the canvas to open the cheat sheet. The ones you'll use most:

| Action | Shortcut |
| --- | --- |
| Save draft | `⌘ S` |
| Undo / Redo | `⌘ Z` / `⌘ Shift Z` |
| Copy / Paste an element | `⌘ C` / `⌘ V` |
| Group / Ungroup | `⌘ G` / `⌘ Shift G` |
| Lock / Unlock | `⌘ L` |
| Zoom in / out | `⌘ +` / `⌘ -` |
| Fit to viewport | `⌘ 0` |
| Select multiple | `Shift` + click |
