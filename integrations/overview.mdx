Carousify covers the LinkedIn side of your stack — drafting, scheduling, analytics, approvals. Integrations are how it talks to everything else: planning content in Notion, getting publish alerts in Slack, and forwarding events through Zapier into anything you already automate.

You don't need any of these to use Carousify. Pick the ones that match a real friction in your workflow.

## Where to find it

In the left sidebar, open **Manage Workspace › Integrations**.

<img
  src="/images/integrations/integrations-page.png"
  alt="Workspace Integrations page showing Notion, Slack, and Zapier with Connect buttons"
  className="rounded-xl border"
/>

Each tile shows an **Active** or **Inactive** status pill and a **Connect** button. Click **Connect** to start the setup flow for that tool.

<Note>
  Integrations are scoped to a single workspace. If you run a workspace per client, connect Notion, Slack, or Zapier inside each workspace separately — settings don't carry across.
</Note>

## Don't see what you need?

The integration list will keep growing. If a tool your team relies on isn't here yet, drop a note via [Support](https://carousify.com/contact-us/) — popular requests move up the roadmap. In the meantime, the [MCP server](/mcp/overview) covers most custom workflows without waiting for a built-in integration.
